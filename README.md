# Hackathon '26 — Back to the Garage

Landing page for e-conomic's 2026 hackathon. Modern layout, 1992-inspired details, MS Paintbrush-style interactive hero where attendees can draw over the event logo.

## Stack

Static HTML/CSS/JS — no build step. Open `index.html` in a browser.

## Features

- Hero with a working Paint app (pencil, brush, eraser, color-aware stamps)
- Brand-color palette and chrome
- Schedule, sign-up, and Q&A sections
- Drawings can be saved as PNG or sent to a Google Drive folder (see `DRIVE_SETUP.md`)

## Deploy

Static site — deploys to Vercel / Netlify / GitHub Pages with no config.
